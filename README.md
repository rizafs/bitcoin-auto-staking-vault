# Bitcoin Auto Staking Vault

A simple DeFi vault on Bitcoin L1 that automatically compounds staking rewards.

## Overview
Bitcoin Auto Staking Vault is an experimental DeFi vault designed to help users maximize yield from Bitcoin-based staking opportunities. The vault automatically collects and compounds rewards, allowing users to earn passive yield without manual intervention.

## Features
- Automatic reward compounding
- Simple vault deposit & withdrawal
- Transparent on-chain logic
- Open source and community driven

## How It Works
1. Users deposit BTC-based assets into the vault.
2. The vault allocates assets to staking opportunities.
3. Rewards are periodically harvested.
4. Rewards are automatically compounded back into the vault.
